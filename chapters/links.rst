
Recommended Books and Websites
==============================


Tutorials and Books
-------------------

- `www.academis.eu <https://www.academis.eu>`_ – tutorials and exercises by **Kristian Rother**
- `Think Python <https://allendowney.github.io/ThinkPython/>`_ – a very systematic, scientific tutorial by **Allen B. Downey**
- `Automate the Boring Stuff with Python <https://automatetheboringstuff.com/>`_ – by **Al Sweigart**
- `Python Basics Tutorial <https://python-basics-tutorial.readthedocs.io/en/latest/>`_ – covers many software engineering tools by **Veit Schiele**

----

Key Libraries
-------------

- `tutorial.djangogirls.org <https://tutorial.djangogirls.org/>`_ – Building an interactive Django website
- `polars go undersea <https://www.academis.eu/polars_go_undersea/>`_ – Data analysis with polars – by **Shreyaasri Prakash and Kristian Rother**
- `pandas go to space <https://www.academis.eu/pandas_go_to_space/>`_ – Data analysis with pandas – by **Kristian Rother**
- `Pixels to generative AI <https://pixels2genai.art>`_ – Creating art with NumPy and neural networks by **Burak Kağan Yılmazer and Kristian Rother**

----

Online Exercises
----------------

- `W3 Schools Python Tutorial <https://www.w3schools.com/python/default.asp>`_ – Lightweight course with exercises
- `www.codewars.com <http://www.codewars.com/>`_ – small online coding tasks
